# 🔍 Key Insights

| #  | Insight                                                                                                        |
| -- | -------------------------------------------------------------------------------------------------------------- |
| 😴 | Finance-related occupations recorded the **lowest average sleep duration**.                                    |
| 👥 | Sleep duration varies across age groups, with participants over 50 showing the highest average sleep duration. |
| ⭐  | Better sleep duration is associated with higher sleep quality.                                                 |
| 😓 | Lower sleep duration is associated with higher stress levels.                                                  |
| 🏃 | Moderate physical activity tends to be associated with better sleep duration.                                  |
| 🌙 | Participants sleeping around **8–8.5 hours** generally reported the lowest stress levels.                      |
| ⚠️ | Participants sleeping **7–8 hours** showed the lowest proportion of sleep disorders.                           |
| 🛌 | Short sleep duration is strongly associated with insomnia.                                                     |
